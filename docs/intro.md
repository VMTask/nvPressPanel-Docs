# 说在前面

nvPress为熊猫开发，本程序为非官方衍生版本，[官网](https://blog.panda-studio.cn/)

## 开始

### 准备工作

- [Ubuntu](https://ubuntu.com) 版本18以上
  - 请确保您的机器有Systemd，否则无法安装
- [CentOS](https://redhat.com) 版本7以上
  - 此系统版本未经过测试，看运气安装成功

两种系统均可安装

### 硬件准备
 - 优良的网络（不是VXN）
 - 内存至少 1h1g
 - 能访问外网
 - 打开22,8080防火墙
### 软件准备
 - Bash
